# Job Tracker

A basic Job application tracker to see what jobs you applied to and the progress.

This currently only tracks Company, Position, Location, On-site/remote, and progress.

I have not or will not use AI generated code for this project. I am using this project to learn without the use of LLMs.

## How to run

### Linux
Run start.sh

### Windows
Run start.bat
